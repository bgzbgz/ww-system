# Requirements Checklist: Boss Office Brand & Design Compliance

**Purpose**: Validate all visual, typographic, and tonal requirements are implemented for Boss Office brand compliance
**Created**: 2026-01-23
**Feature**: [spec.md](../spec.md)

**Note**: This checklist validates the Boss Office UI against Fast Track brand guidelines. Check items off as verified using `[x]`.

---

## User Story 1: Visual Brand Compliance Check (P1)

- [ ] US1-001: Dashboard only uses #000000, #FFFFFF, #B2B2B2, #FFF469 colors
- [ ] US1-002: Tool Detail page only uses approved brand colors
- [ ] US1-003: Upload page only uses approved brand colors
- [ ] US1-004: Preview modal only uses approved brand colors
- [ ] US1-005: All headings use Plaak font in UPPERCASE
- [ ] US1-006: All body text uses Riforma font in sentence case
- [ ] US1-007: All labels use Monument Grotesk Mono in UPPERCASE
- [ ] US1-008: Fast Track F mark visible in top-right header corner

## User Story 2: Typography Hierarchy Compliance (P1)

- [ ] US2-001: Page titles use Plaak, uppercase, bold weight
- [ ] US2-002: Body copy uses Riforma, sentence case, regular weight
- [ ] US2-003: Labels/annotations use Monument Grotesk Mono, uppercase
- [ ] US2-004: Clear visual hierarchy exists between heading levels
- [ ] US2-005: Font sizes are consistent across similar elements
- [ ] US2-006: Line height is comfortable (1.4-1.6 for body text)
- [ ] US2-007: Headlines are short and punchy (not sentences)

## User Story 3: Color System Compliance (P1)

- [ ] US3-001: Page backgrounds are #FFFFFF or #000000 only
- [ ] US3-002: Primary text is #000000 on light, #FFFFFF on dark
- [ ] US3-003: Secondary/muted text uses #B2B2B2 (grey)
- [ ] US3-004: Accent elements use #FFF469 (yellow) sparingly
- [ ] US3-005: No unapproved colors exist (no blues, greens, reds)
- [ ] US3-006: No gradients exist anywhere in the UI
- [ ] US3-007: Yellow is used for emphasis only, not decoration

## User Story 4: Layout and Spacing Compliance (P2)

- [ ] US4-001: Text and elements are strongly left-aligned
- [ ] US4-002: Cards have solid black borders (no subtle greys)
- [ ] US4-003: All corner radii are 0 (sharp corners only)
- [ ] US4-004: Spacing uses consistent, predictable scale
- [ ] US4-005: Desktop layout follows 12-column grid
- [ ] US4-006: White space is generous and intentional

## User Story 5: Forbidden Pattern Detection (P2)

- [ ] US5-001: No gradients exist in the UI
- [ ] US5-002: No rounded corners exist (border-radius: 0)
- [ ] US5-003: No drop shadows exist
- [ ] US5-004: No stock photography or illustrations
- [ ] US5-005: No fancy animations or transitions
- [ ] US5-006: No colored icons (monochrome only)
- [ ] US5-007: No red for errors, no green for success

## User Story 6: Tone of Voice Compliance (P2)

- [ ] US6-001: Button labels use imperative verbs
- [ ] US6-002: Error messages are direct and specific
- [ ] US6-003: Headlines are verdicts, not labels
- [ ] US6-004: No corporate jargon in UI copy
- [ ] US6-005: No hedging words (might, could, perhaps)
- [ ] US6-006: Headings are UPPERCASE, body is sentence case

---

## Visual Requirements (VR-001 to VR-009)

- [ ] VR-001: Color palette limited to #000000, #FFFFFF, #B2B2B2, #FFF469
- [ ] VR-002: No colors outside the approved palette
- [ ] VR-003: Yellow used sparingly for emphasis only
- [ ] VR-004: Backgrounds are only #FFFFFF or #000000
- [ ] VR-005: All borders are solid black, minimum 2px
- [ ] VR-006: Border radius is 0 on all elements
- [ ] VR-007: No gradients exist
- [ ] VR-008: No drop shadows exist
- [ ] VR-009: No transparency effects (except overlays/modals)

## Typography Requirements (TR-001 to TR-007)

- [ ] TR-001: Headlines use Plaak Ex Condensed 43 Bold, UPPERCASE
- [ ] TR-002: Body copy uses Riforma LL Regular, sentence case
- [ ] TR-003: Labels use Monument Grotesk Mono, UPPERCASE
- [ ] TR-004: Fonts loaded from Designs/03. Fonts/woff2/ (WOFF2)
- [ ] TR-005: Font stack includes system fallbacks
- [ ] TR-006: Line height is 1.4-1.6 for body text
- [ ] TR-007: Headlines are short, punchy statements

## Layout Requirements (LR-001 to LR-006)

- [ ] LR-001: Text and elements are strongly left-aligned
- [ ] LR-002: Desktop uses 12-column grid system
- [ ] LR-003: Cards have solid black borders (3px minimum)
- [ ] LR-004: Spacing uses 8px base unit scale
- [ ] LR-005: Visual hierarchy is clear through size/weight/spacing
- [ ] LR-006: White space is generous and intentional

## Logo Requirements (LOGO-001 to LOGO-004)

- [ ] LOGO-001: F mark appears in top-right corner of header
- [ ] LOGO-002: F mark uses white version (FastTrack_F_White.png)
- [ ] LOGO-003: F mark has minimum 16px padding from edges
- [ ] LOGO-004: F mark is not distorted, recolored, or animated

## Button Requirements (BTN-001 to BTN-006)

- [ ] BTN-001: Primary CTAs have yellow background (#FFF469), black text
- [ ] BTN-002: Secondary buttons have black background, white text
- [ ] BTN-003: Tertiary buttons have transparent bg, black border/text
- [ ] BTN-004: Hover states are simple (invert or underline)
- [ ] BTN-005: Focus states visible with yellow outline
- [ ] BTN-006: Disabled states use grey (#B2B2B2)

## Tone of Voice Requirements (TOV-001 to TOV-007)

- [ ] TOV-001: Buttons use imperative verbs (Upload, Approve, Reject)
- [ ] TOV-002: Headlines are verdicts, not labels
- [ ] TOV-003: Errors are direct, never apologetic
- [ ] TOV-004: No corporate jargon (synergy, leverage, optimize)
- [ ] TOV-005: No hedging words (might, could, perhaps, possibly)
- [ ] TOV-006: Copy is concise (one idea per sentence)
- [ ] TOV-007: Numbers only when citing real data

## Forbidden Patterns (FP-001 to FP-011)

- [ ] FP-001: No rounded corners anywhere
- [ ] FP-002: No gradients anywhere
- [ ] FP-003: No drop shadows anywhere
- [ ] FP-004: No generic stock photography
- [ ] FP-005: No illustrations or CGI
- [ ] FP-006: No colored icons (monochrome only)
- [ ] FP-007: No fancy animations or transitions
- [ ] FP-008: No red for errors, no green for success
- [ ] FP-009: No centered text (except intentional heroes)
- [ ] FP-010: No soft, muted, or pastel colors
- [ ] FP-011: No corporate buzzwords in copy
